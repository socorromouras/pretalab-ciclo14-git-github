Tarefas diárias - Fluxo Git

1. git pull origin main - Atualizar repositório local

2. git status - Verificar estado atual

3. Criar nova branch para feature: git checkout -b feature/nova-funcionalidade

4. Desenvolver novas funcionalidades 

5. git add . - Adicionar alterações

6. git commit -m "mensagem clara do que foi feito" - Commitar mudanças

7. Repetir passos 4-6 conforme necessário

8. git push origin feature/nova-funcionalidade - Enviar para repositório remoto

9. Criar Pull Request no GitHub

10. git checkout main - Voltar para branch principal

11. git pull origin main - Atualizar local com últimas mudança
